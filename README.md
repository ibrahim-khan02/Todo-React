#  Todo React App

A simple and clean Todo application built using React (Vite).  
It allows users to add, delete, and manage tasks easily.

---
Features
Add new tasks
Delete tasks
Mark tasks as completed (if implemented)
Fast and responsive UI

---

##  Tech Stack
- React
- Vite
- JavaScript 
- CSS
